WWP Namespaces

From WikiBase
Jump to navigation Jump to search

This page documents the use of XML namespaces at the WWP. Its most important function is to provide a list of namespaces (i.e., URIs) that we use, thus to prevent duplication of an existing namespace. But we also include the prefix we commonly use and a short blurb with each.

WWP
[nil]
http://www.wwp.northeastern.edu/ns/textbase is the main namespace for the WWO textbase; it is essentially our local replacement for http://www.tei-c.org/ns/1.0
wms
http://www.wwp.northeastern.edu/ns/textbase/manuscript is the namespace for extra-WWO manuscript encoding. Currently used almost exclusively by the Mary Moody Emerson project
wf
http://www.wwp.northeastern.edu/ns/functions is the namespace we use for XSLT and XQuery functions we write
wwp
http://www.wwp.northeastern.edu/ns/internal_documentation/1.0 is the namespace used for the internal documentation system, except for its examples
eg
http://www.wwp.northeastern.edu/ns/internal_documentation/Examples is the namespace for examples within the internal documentation system
wex
http://www.wwp.neu.edu/ns/exhibits/1.0 is the namespace for exhibits
none
http://www.wwp.northeastern.edu/ns/programming is never used (it is declared in MMEphase2.xslt, but not used)
wq
http://www.wwp.northeastern.edu/ns/quotes is the namespace used for the few constructs added to TEI for the quote of the day system (i.e., our quotography)
wwp
http://www.wwp.northeastern.edu/ns/programs was used in generate_tadpoles for what I think we now use http://www.wwp.northeastern.edu/ns/functions
YAPS
yaps
http://www.wwp.northeastern.edu/ns/yaps is the namespace for Yet Another Presentation System
others
[nil]
http://www.wwp.neu.edu/temp/ns temporary pedigogical or for temporary structures internal to a program
