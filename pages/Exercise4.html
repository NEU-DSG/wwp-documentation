For this exercise, we're revisiting poetry and adding some additional features. Use the second poem example ("File:Poems smith2.pdf"). You can either create a new test document using the training template, or you can just add another <div> element to your existing poem transcription.

Here are the special features you'll need to pay attention to:

  • stanzas: you can read more about how to encode these in the discussion of <lg> in the WWP documentation
  • quotations that cross element boundaries (ask Syd or Julia if we haven't covered this yet).
